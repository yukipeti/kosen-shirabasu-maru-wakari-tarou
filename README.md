# kosen-shirabasu-maru-wakari-tarou

A simple interactive prompt utility.

## Usage

Run the prompt utility:

```bash
python3 prompt.py
```

The prompt will accept user input and echo it back. Type `exit` or `quit` to exit the prompt.

## Requirements

- Python 3.x

## Example

```
$ python3 prompt.py
Welcome to Kosen Shirabasu Maru Wakari Tarou Prompt!
Type 'exit' to quit

prompt> hello
You entered: hello
prompt> exit
Goodbye!
```
